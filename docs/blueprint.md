# **App Name**: Desi Content Creator

## Core Features:

- Instagram Caption Generator: Generate trendy Instagram captions in Hinglish based on user input topics.
- YouTube Title Generator: Generate click-worthy YouTube titles and SEO-optimized descriptions for videos on Indian-related topics.
- Instagram Bio Generator: Create short and engaging Instagram bios in Hinglish tailored to the user's brand or personality.
- Hashtag Generator: Generate relevant hashtags (high reach + low competition) for Indian content creators.
- Inspirational Quote Generator: Generate short inspirational and devotional quotes in simple Hindi for social media sharing.
- Content Topic Suggestion Tool: Suggest content topics suitable for social media, relevant to an Indian audience, based on current trends.
- User Preferences Storage: Allow users to save their preferred language styles (Hinglish/Hindi), tone (viral/clean), and frequently used hashtags.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to reflect Indian culture and energy.
- Background color: Very light beige (#F5F5DC) to provide a neutral canvas.
- Accent color: Deep sky blue (#00BFFF) for call-to-action buttons.
- Font: 'Poppins' (sans-serif) for a contemporary and readable interface, suitable for short captions and titles.
- Use relevant Indian-themed icons and emojis throughout the app.
- Clean and modern layout with clear sections for each content generation tool.
- Subtle animations for generating content, providing a delightful user experience.
